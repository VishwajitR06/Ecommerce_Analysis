# 📦 E-Commerce Sales Dashboard

## 📌 Overview
This project is an interactive **E-Commerce Sales Dashboard** built using **Streamlit, Pandas, and Matplotlib**.  
It allows users to upload a dataset and visualize key insights such as sales distribution, country-wise performance, and transaction details.

---

## 🚀 Features
- 📂 Upload CSV dataset dynamically  
- 📊 View raw data and summary statistics  
- 🌍 Analyze total quantity sold by country  
- 🔍 Filter data based on selected country  
- 📈 Interactive visualizations using Matplotlib  

---

## 🛠️ Tech Stack
- Python  
- Streamlit  
- Pandas  
- Matplotlib

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository
```
git clone https://github.com/your-username/ecommerce-app.git
cd ecommerce-app
```

### 2️⃣ Create virtual environment (optional but recommended)
```
python -m venv .venv
.venv\Scripts\activate
```

### 3️⃣ Install dependencies
```
pip install -r requirements.txt
```

---

## ▶️ Run the Application
```
streamlit run app.py
```

Then open in browser:
```
http://localhost:8501
```

## 📊 Dataset Information

  - Country

Due to GitHub file size limitations, the full dataset is not included.
